> [!IMPORTANT]
> Built for IT teams using the Autotask and Datto RMM platforms.

## Features

- **AI Assistant** — Summarize tickets and clean up note formatting using ChatGPT, Gemini, or Claude. Requires your own API key.
- **Response Templates** — Create reusable email templates with dynamic variables like `${loggedinUser}`, `${ticketPriority}`, `${currentDate}`, and more.
- **Password Generator** — Generate secure, customizable passwords.
- **TOTP Generator** — Create TOTP keys in both Hex and Base32, useful for Duo Security and IT Glue.
- **Incognito URL Rules** — Automatically open specified URLs (e.g. Azure, Microsoft 365, Google Admin) in incognito mode.
- **Datto RMM Enhancements** — OTP injection in the RMM remote session sidebar, True Stretch for remote sessions and cache IT Glue passwords.

## Manual installation

1. [Download](https://github.com/El3ctr1cR/SupportTools-extension/archive/refs/heads/main.zip) the latest release.
2. Go to `chrome://extensions/` and enable **Developer mode**.
3. Click **Load unpacked** and select the extension folder.
